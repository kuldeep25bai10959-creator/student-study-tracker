# Student Study Tracker

## Project Overview
Student Study Tracker is a simple Python-based project that helps students track their study sessions and calculate the total number of hours studied.

## Problem Statement
Many students do not track their study time, which makes it difficult to evaluate productivity and manage time effectively.

## Objective
The objective of this project is to create a simple system that allows students to record their study sessions and monitor their total study hours.

## Features
- Add study sessions
- Record subject and study hours
- Calculate total study time
- Store study data in a file

## Technologies Used
- Python
- JSON file handling
- GitHub for version control

## How the Program Works
1. The user runs the program.
2. The program asks for a menu option.
3. The user can add a study session by entering subject and hours.
4. The program stores this data in a JSON file.
5. The user can also view the total study hours.
